# parcel-project Frontend

## Development

### Running the Project

Start development environment running

```sh
$ yarn start

| `yarn <script>`       | Description                                                                                                             |
| --------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| `start`               | Serves your app at `localhost:1234`                                                                                     |
| `build`               | Builds the application           |

## Production

Generate production files running

```sh
$ yarn build
# or with npm
$ npm run build
```
